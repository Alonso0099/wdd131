Site Planning Document

Site Name

PillWise

The name "PillWise" represents a smart and simple way to manage medications and build better daily habits.

Site Purpose

PillWise helps users track their medications, monitor pill counts, and estimate refill timing. It also provides general information about common medications.

Scenarios

  • How can I keep track of how many pills I have left?
  • When should I refill my medication?
  • What is this medication commonly used for?

Color Schema

  • Dark Blue (#1F3A5F) – headers and navigation
  • Teal (#4FB6A6) – buttons and highlights
  • Light Gray (#F5F7FA) – background
  • Dark Gray (#333333) – text

Typography

  • Poppins – headings
  • Roboto – body text

Wireframes

Mobile View

Stacked layout with navigation menu, content sections, and cards displayed vertically.

Desktop View

Horizontal navigation with grid-based layout and multiple cards displayed side by side.
